## 1. Dependencies
### Install python3
```bash
$ brew install python3
```


### Install ffmpeg
  ```bash
$ brew install ffmpeg
  ```


### Install numpy, aeneas, jupyter lab using pip
```bash
$ pip3 install numpy,aeneas,jupyterlab
```  
    
## 2. Run jupyter lab
```bash
$ jupyter lab
```  
On jupyter lab page, open `audio_split.ipynb` and follow steps from there
